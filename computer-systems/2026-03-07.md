# Sistemas Operacionais

<br>

## 📌 Sistemas Operacionais

- Conjunto de rotinas.
- Gerenciamento dos processos do sistema.
- Interface entre usuário e computador.

### Sem Sistema Operacional
- Necessidade de grande conhecimento de hardware.
- Controle direto do equipamento pelo usuário.

<br>

## 📌 Conceito

- Software que gerencia os aplicativos do computador.
- Interface entre usuário e máquina.

### Objetivo
- Interface simples.
- Execução rápida.
- Operação segura.

<br>

## ⚙️ Sistema Operacional x Programa

- Execução não linear.
- Rotinas executadas concorrentemente.

### Eventos Assíncronos
- Eventos que ocorrem a qualquer momento.
- Execução dependente de eventos.

<br>

## ⚙️ Funções do Sistema Operacional

- Conjunto de rotinas específicas.

### Gerenciamento de Dados
- Leitura e gravação em USB ou disco.
- Posicionamento de dados em trilha e setor.
- Transferência de dados para a memória.
- Informar conclusão de operações.

### Compartilhamento de Recursos
- Controle de acessos em sistemas com vários usuários.

<br>

## 📌 Máquina de Camadas

### Conceito
- Estrutura em níveis.
- Níveis dependem da profundidade de uso.

### Níveis
- Hardware → nível base.
- Sistema Operacional → nível superior.
- Podem existir mais níveis.

### Implementação
- Operações podem ser implementadas em hardware ou software.
- Decisão depende do projeto do sistema.
- Aspectos considerados:
  - custo
  - confiabilidade
  - desempenho

### Integração
- Hardware e software funcionam de forma integrada.
- Unidade lógica para o usuário.

<br>

## 📌 Evolução das Máquinas

### Primeiros Computadores
- Programação em linguagem de máquina.
- Uso de painéis de fios.
- Grande conhecimento de hardware necessário.
- Sistema Operacional simplificou a interação usuário-computador.

<br>

## 📌 Década de 1940

- Segunda Guerra Mundial impulsionou o desenvolvimento de computadores.
- Primeiros computadores eletrônicos utilizavam válvulas.
- Máquinas grandes, lentas e de confiabilidade limitada.

## 💡 Observações

- O Sistema Operacional abstrai a complexidade do hardware.
- Permite que vários programas utilizem os mesmos recursos.
- É essencial para segurança e gerenciamento do sistema.

<br>

## ❗ Atenção

- Sistema Operacional não é apenas interface gráfica.
- Hardware sem software tem uso extremamente limitado.
- Falhas no SO podem comprometer todo o sistema.
