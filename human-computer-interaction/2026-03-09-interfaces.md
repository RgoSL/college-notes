# Interface Humano-Computador 

<br>

## 📌 Conceitos Principais

### Conceito de Interface

* Fronteira entre humano e máquina
* Ponto de encontro entre dois ecossistemas
* Conecta intenção humana ao sistema digital

### Função da Interface

* Traduz o modelo mental humano
* Converte intenção em comandos para o sistema
* Permite execução das ações desejadas

### Significação

* Pistas visuais que indicam interação
* Mostra onde e como interagir
* Exemplos: hover em elementos
* Exemplos: botão vermelho de desligar

### Affordance

* Qualidade de interação intuitiva
* Interação natural facilita o uso
* Quanto mais natural, melhor o affordance

<br>

## ⚙️ Verbos da Interação Humano-Computador

### Instruir

* Envio de comandos diretos
* Usuário fornece instruções ao sistema
* Exige conhecimento prévio

### Conversar

* Uso de linguagem natural
* Comunicação verbal com o sistema
* Interação baseada em diálogo

### Manipular

* Interação com objetos digitais
* Arrastar elementos
* Clicar
* Soltar objetos

### Explorar

* Navegação em ambientes espaciais
* Uso de gestos
* Interface como espaço de interação
* Menor dependência de periféricos

<br>

## 📌 UI Baseada no Mundo Real

### Interface Gráfica

* Baseada em metáforas do mundo real
* Interação visual direta
* Manipulação de elementos digitais

### Interação Natural

* Arrastar elementos
* Clicar em elementos
* Soltar objetos digitais

### Relação com Affordance

* Quanto mais realista o digital
* Mais intuitiva a interação
* Maior facilidade de uso

<br>

## 📌 Evolução Histórica das Interfaces

### Era das Máquinas (1940–1960)

* Sem interação em tempo real
* Comunicação física com a máquina
* Foco em hardware
* Processamento em lote

### Revolução do PC (1970–1990)

* Surgimento da interface gráfica
* Interação visual direta
* Uso de mouse e tela

### Era Mobile (2000–2010)

* Popularização dos smartphones
* Interação por toque
* Uso imediato
* Menor uso de mouse

### Futuro das Interfaces

* Interfaces invisíveis
* Uso de inteligência artificial
* Sensores contextuais
* Interfaces neurais
* Interação sem telas

<br>

## 📌 Paradigmas de Interação

### Sem Interface

* Input manual preparado pelo usuário
* Processamento mecânico
* Output posterior

### Interface Gráfica

* Interação em tempo real
* Manipulação visual direta
* Cursor como extensão da mão

### Interface de Toque

* Interação direta com conteúdo
* Menos intermediários físicos

### Interfaces Futuras

* Interfaces invisíveis
* Interação preditiva
* Integração com IA
* Uso de BCI (Brain Computer Interface)

<br>

## 💡 Observações Importantes

* Interfaces boas reduzem esforço cognitivo
* Metáforas do mundo real aumentam compreensão
* Intuitividade depende do contexto do usuário

<br>

## ❗ Atenção

* Nem toda interface “moderna” é intuitiva
* Excesso de elementos prejudica usabilidade
* Falta de feedback visual confunde o usuário
